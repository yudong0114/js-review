## セクション

### 画像をアップロード＆プレビューしよう
* `event`には`value、text、files`などがある
* `preventDefault()`で要素のデフォルトの動作を無効(ex. `file`でfinderが開くなど)
* `stopPropagation()`で子要素のイベントが親要素に影響を与えないようにする(伝播を防ぐ)
